# Resender TG Bot

Telegram-бот для отложенного постинга медиа (фото/видео/альбомы) в канал. Управление — полностью через Telegram.

## Возможности

- Приём фото, видео и альбомов — автоматическая буферизация `media_group`
- **Instagram-ссылки** — отправьте ссылку на Reels или пост, бот скачает медиа (через yt-dlp) и добавит в очередь
- Очередь постов с пагинацией и inline-кнопками (перемещение, удаление)
- **Быстрое удаление** — кнопка «🗑 Удалить» сразу после добавления в очередь
- Глобальная подпись (HTML) — применяется ко всем публикациям
- Два режима расписания:
  - **Слоты** — конкретные часы с опциональной привязкой к дням недели
  - **Интервал** — каждые N минут
- Публикация по расписанию (APScheduler) или вручную по команде
- Пауза/возобновление публикаций
- Восстановление пропущенных постов после рестарта (максимум 1)
- Статистика с форматированными датами
- **Постоянная клавиатура** — цветные кнопки «📋 Очередь» и «📅 Расписание» всегда на виду
- Цветные inline-кнопки (`style`: danger/success/primary — Bot API 9.4)
- `date_time` entity для отображения времени в часовом поясе пользователя (Bot API 9.5)

## Стек

| Компонент | Технология |
|-----------|-----------|
| Язык | Python 3.11+ |
| Telegram | aiogram 3.x (Bot API 9.5) |
| БД | PostgreSQL + SQLAlchemy 2.x (async, asyncpg) |
| Миграции | Alembic |
| Планировщик | APScheduler 4.x |
| Конфигурация | pydantic-settings |
| Скачивание медиа | yt-dlp + ffmpeg |
| Деплой | Docker / Railway |

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и инструкция |
| `/help` | Справка по командам |
| `/caption` | Показать/изменить подпись |
| `/caption <текст>` | Задать новую подпись (HTML) |
| `/queue` | Очередь постов с управлением |
| `/schedule` | Настройка расписания (слоты/интервал) |
| `/next` | Следующий пост и время публикации |
| `/publish` | Опубликовать следующий пост сейчас |
| `/pause` | Пауза публикаций |
| `/resume` | Возобновить публикации |
| `/stats` | Статистика |

Отправьте фото, видео или ссылку на Instagram Reels/пост — бот добавит в очередь автоматически.

## Быстрый старт

### 1. Клонирование

```bash
git clone https://github.com/porchax/resender-tg-bot.git
cd resender-tg-bot
```

### 2. Настройка окружения

```bash
cp .env.example .env
```

Заполните `.env`:

```
BOT_TOKEN=ваш_токен_бота
ADMIN_ID=ваш_telegram_id
CHANNEL_ID=id_канала (например -1001234567890)
DATABASE_URL=postgresql+asyncpg://postgres:postgres@localhost:5432/resender
TIMEZONE=Europe/Moscow
```

### 3. Запуск с Docker Compose

```bash
docker-compose up --build
```

Бот поднимет PostgreSQL, применит миграции и запустится.

### 4. Локальная разработка (без Docker)

```bash
pip install .
# Убедитесь что PostgreSQL запущен и DATABASE_URL указан
alembic upgrade head
python -m bot
```

## Деплой на Railway

1. Создайте проект на Railway
2. Добавьте PostgreSQL plugin — `DATABASE_URL` установится автоматически
3. Задайте переменные: `BOT_TOKEN`, `ADMIN_ID`, `CHANNEL_ID`, `TIMEZONE`
4. Railway подхватит `Dockerfile` и задеплоит

> Бот автоматически конвертирует `postgres://` и `postgresql://` в `postgresql+asyncpg://`.
